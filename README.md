# simple-to-do-list
Features
View Tasks: Display all tasks with their current completion status.

Add Tasks: Quickly add new items to your list.

Mark Completion: Toggle tasks as completed with a checkmark symbol.

Remove Tasks: Delete specific tasks from the list by their index number.

Input Validation: Basic checks to ensure valid task selection.
